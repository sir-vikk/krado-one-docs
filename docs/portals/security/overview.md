# Security Overview

**Route:** `/security` · **Edition:** Pro+ · **Beta**

The Security Overview summarises your current vulnerability posture, secrets hygiene, and compliance status at a glance.

![Security Overview](/assets/screenshots/security/overview.png)

## Summary cards

| Card | Description |
|------|-------------|
| Open vulnerabilities | Total CVEs in Open/In-Progress status |
| Critical CVEs | Subset with CVSS score ≥ 9.0 |
| Secrets tracked | Total secrets in the inventory |
| Compliance controls | Passed vs total controls across all frameworks |

## Sections

Navigate to each sub-section using the left sidebar:
- **Vulnerabilities** — full CVE list with status tracking
- **Secrets** — secrets inventory and rotation status
- **Compliance** — framework controls and evidence mapping
