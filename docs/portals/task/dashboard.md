# Task Dashboard

The Task Dashboard is your personal work hub in Krado One. It provides a kanban board for organizing tasks, quick access to JIRA integration, and a clear view of what needs your attention.

## How to Access

Navigate to **Task Portal** from the left sidebar. The dashboard loads as the default view.

## Using the Kanban Board

Tasks are organized into four columns representing workflow stages:

- **To Do** -- tasks that have not been started
- **In Progress** -- tasks currently being worked on
- **In Review** -- tasks awaiting review or approval
- **Done** -- completed tasks

Drag cards between columns to update their status instantly. The board provides a visual overview of your current workload and team progress.

## Creating a Task

1. Click **+ New Task** in the top-right corner of the dashboard.
2. Fill in the task details:
    - **Title** -- a brief description of the task
    - **Description** -- additional context or requirements
    - **Priority** -- Critical, High, Medium, or Low
    - **Due Date** -- the target completion date
    - **Assignee** -- the person responsible for the task
    - **Team Label** -- the team this task belongs to
3. Click **Save** to add the task to the board.

The new task appears in the **To Do** column by default.

## Managing Tasks

| Action | How to Do It |
|--------|-------------|
| Move a task | Drag the card to the target column |
| Edit a task | Click the card title to open the detail view |
| Delete a task | Open the card, click the overflow menu, and select Delete |
| Filter by priority | Use the priority filter in the toolbar |
| Filter by due date | Use the date range picker in the toolbar |

## Understanding Task Cards

Each card on the board displays:

- Title and a description preview
- Priority badge (color-coded by severity)
- Due date (highlighted in red if overdue)
- JIRA ticket link (if the task is synced with JIRA)
- Assignee and team label

## Stats Bar

A summary row at the top of the dashboard shows counts for open, in-progress, and overdue tasks, giving you a quick health check without scanning the entire board.
