# Tech Radar

**Route:** `/platform/tech-radar` · **Edition:** Pro+ · **Beta**

The Tech Radar tracks your organisation's technology adoption stance across four quadrants: Languages & Frameworks, Platforms, Tools, and Techniques.

![Tech Radar](/assets/screenshots/platform/tech-radar.png)

## Quadrants

| Quadrant | Examples |
|----------|---------|
| **Languages & Frameworks** | Go, React, Python, gRPC |
| **Platforms** | Kubernetes, AWS EKS, Cloudflare |
| **Tools** | Terraform, ArgoCD, Datadog |
| **Techniques** | GitOps, trunk-based development, SLO-based alerting |

## Rings

Each item is placed in one of four rings:

| Ring | Meaning |
|------|---------|
| **Adopt** | Recommended — use broadly |
| **Trial** | Worth exploring on a project |
| **Assess** | Investigate, not ready for production use |
| **Hold** | Not recommended — phase out or avoid |

## Adding items

Click **Add Item** and specify the name, quadrant, ring, and a description of why it's positioned there. Items can be updated as your stance evolves.
