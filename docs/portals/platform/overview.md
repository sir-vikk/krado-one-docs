# Platform Engineering Overview

**Route:** `/platform` · **Edition:** Pro+ · **Beta**

The Platform Engineering Portal consolidates tooling for the platform team: tech radar, CI/CD pipeline tracking, API catalog, change management, environment promotions, status pages, and developer onboarding.

![Platform Engineering Overview](/assets/screenshots/platform/overview.png)

## Sections

| Section | Route | Description |
|---------|-------|-------------|
| Tech Radar | `/platform/tech-radar` | Technology adoption tracker |
| CI/CD Pipelines | `/platform/pipelines` | Pipeline run history and stats |
| API Catalog | `/platform/api-catalog` | Internal API registry |
| Change Requests | `/platform/change-requests` | Change approval workflow |
| Env Promotions | `/platform/promotions` | Environment promotion tracking |
| Status Page | `/platform/status-page` | Public-facing service status |
| Onboarding | `/platform/onboarding` | Developer onboarding checklists |
