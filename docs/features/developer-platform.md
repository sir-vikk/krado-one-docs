# Developer portal modules

The Enterprise UI exposes (subject to feature flags and edition) modules backed by routes such as:

- **Service catalog**, **golden path / scaffolding**, **GitHub / Terraform**  
- **Metrics / DORA**, **scorecards**, **AI assistant**  
- **Runbooks**, **MCP servers**, **executions**  
- **Cloud** inventory, **FinOps**, **compliance**  
- **Tech radar**, **pipelines**, **secrets**, **change requests**, **API catalog**, **vulnerabilities**, **status page**, **promotions**, **onboarding**, **plugins**, **audit**

Use the [API explorer](/api-explorer) as the canonical route index. Edition requirements are enforced per route group in **`RegisterRoutes`**.
