Setup

Running in under 5 minutes

One Docker Compose file. No cloud account, no build toolchain on your host.

1
Pull the image

Single Docker Compose file — backend, frontend, and database in one command.

2
Run the setup wizard

Create your org and admin account on first launch — under 60 seconds.

3
Connect your tools

Add Jira, Slack, AWS via env vars or the Portal Config UI. No YAML files.

4
Invite your team

Add members, assign roles, toggle the portals your team needs.

terminal
$curl -fsSL https://get.krado.one | sh
✓ Downloaded docker-compose.yml
$docker compose up -d
✓ backend → healthy
✓ frontend → healthy
✓ database → healthy
$open http://localhost
🚀 Krado One is ready at http://localhost
