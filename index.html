How it works

Running in under 5 minutes

One compose file. No cloud account required, no build toolchain on your host.

1
Pull the image

Single Docker compose file. Backend, frontend, and database in one command.

2
Run the setup wizard

Create your org and admin account on first launch. Takes under 60 seconds.

3
Connect your tools

Add Jira, Slack, GitHub, AWS via env vars or the Portal Config UI — no YAML files.

4
Invite your team

Add members, assign roles, and toggle the portals your team needs.

# grab the compose file
curl -O https://raw.githubusercontent.com/

# start everything
docker compose up -d
# open in browser
